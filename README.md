# FlowGame
Simple game made for understanding MVC pattern in Java.
## Overview
Based on Java SWING for display and GSON for loading level
## Level format

     {  
          "goals": [  
          {  
          "type": "S1",  
          "coords": [  
          {  
          "X": 4,  
          "Y": 4  
          }
          ]
          ]  
        }
